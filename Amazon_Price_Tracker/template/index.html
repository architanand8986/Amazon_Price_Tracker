    Welcome To the Amazon Price Tracker

    This projcet helps the user to track the prices of the products availaible at the amazon site.This project keeps track of the latest 7 dates tracked by the user and present it to the user in the table form.Down below are the procedures on how to use this site in an effective and proper way.

    Instructions

    1.Create Account

    Click on the Accounts in the navbar of the site and scroll-down on the link to go to the sign-up page. Fill the form accordingly and your account is created

    2.Add Products

    After your account is created, sign-up and you will be directed to the page for tracking item. Fill the form with correct amazon page url of the product

    3.See Product Activity

    You would be directed to the product page. You will see the details of your previous trackers here.You can update the name of the product using Update button, delete product tracker completely with Delete button and see the tracker activity from Activity button

    4.See Tracker Activity

    In tracker activity page,the user would see the table with date and prices columns and seven rows. For tracking the price of elements click on the track button on the top-right area.The User can track data for a product only once a day

    5.Add new Tracker

    Below the table,you can see the track button, click on it to track the object activity, User can track an object only once a day, if no data can be extracted from the url then please check the validity of the URL

    6.Advanced Analysis

    This feature gives graphical plot of the price activity as compared to their median and mean price. This activity can be accessed after the user has tracked the object at least 3 times.
